# Maintain  
  
In diesem Tab können die Updates für Tasmota, TFT-Firmware und Script durchgeführt werden.
Wie im Bild zu sehen, steht hinter dem Panelnamen **"Überprüfen"**, dies bedeutet das ein Update für das Panel verfügbar ist.
Durch klick auf den Balken, öffnet sich der untere Bereich und die Aktualisierungen werden sichtbar.  Jetzt kann über die Button das Update angestoßen werden. Wenn das Update durchgelaufen ist, sollte der Button **Tabelle aktualisieren** gedrückt werden, um zu Prüfen das alle Meldungen wieder in Ordnung sind.   
  
> [!Warning]  
> Wenn oben der rote Schriftzug **Beta-Version in Verwendung** zu lesen ist, nutzt ihr von uns freigegebende Beta-Version. Die Umstellung erfolgt im Tab **Global-Setting**. Dazu muss der Experten-Modus des ioBroker aktiviert sein, damit ihr die Einstellung sehen könnt.  
  
> [!Note]  
> Es können hier nur Versionen installiert werden, die durch uns freigegeben wurden.
