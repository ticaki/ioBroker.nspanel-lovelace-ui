# Symbol Übersicht
  
In diesem Tab werden alle Symbole dargestellt, die auf dem Panel angezeigt werden können.  
Mit Strg+f (Win/Linux) oder Cmd+f (Mac) wird ein Filter-Feld eingeblendet. Wenn das Sybol gefunden wurde, wird mit einem Klick auf das Symbol der Name in die Zwischenablage kopiert.
